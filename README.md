# Development

Requirements:

- [Rust](https://www.rust-lang.org/) and [Tauri](https://tauri.app/)
- [Node.js](https://nodejs.org/en) and [Yarn](https://yarnpkg.com/)

Run App in debug mode:

```shell
yarn tauri dev
```

Build:

```shell
yarn tauri build
```
